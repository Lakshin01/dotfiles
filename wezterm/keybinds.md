| Action             | Keys                 |
| ------------------ | -------------------- |
| Split Vertical     | `CTRL + ALT + v`     |
| Split Horizontal   | `CTRL + ALT + h`     |
| Move Between Panes | `CTRL + h/j/k/l`     |
| Close Pane         | `CTRL + ALT + x`     |
| New Tab            | `CTRL + ALT + t`     |
| Next Tab           | `CTRL + Tab`         |
| Previous Tab       | `CTRL + SHIFT + Tab` |
